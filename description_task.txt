Задача: Учет товарных запасов
Описание задачи:

Вам необходимо создать программу для учета товарных запасов на складе. Программа должна выполнять следующие функции:

Позволить пользователю ввести данные о товаре (название, количество, цена за единицу).
Рассчитать общую стоимость каждого товара (цена за единицу * количество).
Рассчитать общую стоимость всех товаров на складе.
Вывести отчет с деталями о каждом товаре, включая его название, количество, цену за единицу и общую стоимость.
После ввода всех товаров, вывести общую стоимость всех товаров на складе.

Условия задачи:

Программа должна запрашивать данные о товаре в цикле.
Для каждого товара вводятся: название, количество и цена за единицу.
Пользователь вводит данные для нескольких товаров и завершает ввод с помощью команды стоп.
Программа должна вывести:
Список всех товаров с их количеством, ценой за единицу и общей стоимостью.
Общую стоимость всех товаров на складе.


Для расчета общей стоимости каждого товара использовать формулу:
Общая стоимость товара = количество * цена за единицу.
